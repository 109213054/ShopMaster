軟體工程期中專題:購物車平台
[客戶端功能]
1.列出尚有庫存之貨品清單
2.將貨品加入購物車
3.列出/刪除購物車內之貨品項目
4.將購物車內容結帳
[管理端功能]
1.增刪修改貨品項目
2.增加商品庫存(商品入庫)
3.已結帳購物車出貨(商品出貨改庫存)
[開發面要求]
1.需使用MVC架構，需使用git作版本管理(提供github專案連結給助教)
2.需繳交簡要技術文件 (DB Schema, 系統架構說明，直接上github)
3.完成後跟助教約時間demo
4.助教會提一項修改需求，要能修改完成才算完成此專案


[檔案說明]
home.html:使用者登入介面，區分誰是管理者/消費者

MListItem.py: 管理商品的介面，有商品的清單給管理者進行新增、刪除、更新和出貨的介面
MaddItem.py: 管理者送出表單後，就會來這個檔案執行新增新商品的功能
MdelItem.py: 管理者送出表單後，就會來這個檔案執行刪除商品的功能
Maddinventory.py: 管理者送出表單後，就會來這個檔案執行更新商品庫存的功能
Mout.py: 給管理者出貨的介面，裡面有出貨列表，管理者可以選擇要一次全部出貨，還是將指定的商品出貨
Mpay.py: 管理者選擇一次全部出貨後，就會來這個檔案執行
MeachOut.py: 管理者選擇個別出貨後，就會來這個檔案執行

CListItem.py: 消費者要購物的介面，如果有要買的商品，可以透過加入購物車的介面輸入想買的商品
CaddCart.py: 消費者送出表單後，就會來這個檔案執行加入購物車的功能
CgetCartItem.py: 購物車介面，提供消費者更改購買數量、取消購買及結帳的功能
CeditCart.py:消費者送出表單後，就會來這個檔案執行更新購買數量的功能
CdelCart.py: 消費者送出表單後，就會來這個檔案執行取消購買數量的功能
Cout.py: 確認結帳的介面，再次向消費者確認要不要結帳
Cpay.py:消費者確認結帳後，就會來這個檔案執行結帳的功能

mitemlist.sql:資料庫的匯出檔
dbConfig:連接資料庫
mitemlist.py: 以上檔案需要用的函式都寫在這
